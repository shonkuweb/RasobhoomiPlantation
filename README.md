## ğŸ› ï¸ Setup & Installation

### Prerequisites
- Node.js (v18+)
- npm

### Local Development

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Environment Setup**
    Create a `.env` file in the root directory:
    ```env
    PORT=3000
    NODE_ENV=development
    PHONEPE_MERCHANT_ID=your_id
    PHONEPE_SALT_KEY=your_key
    ADMIN_PASSCODE=1234  # Change this!
    ```

3.  **Run Development Server**
    ```bash
    npm run dev:all
    ```
    This runs both the backend server and Vite frontend concurrently.

4.  **Build for Production**
    ```bash
    npm run build
    ```

## ğŸš¢ Deployment (Hostinger VPS / Docker)

1.  **SSH into your VPS**.
2.  **Clone/Upload** your project.
3.  **Configure `.env`** for production (ensure `DB_TYPE=postgres` and secure `ADMIN_PASSCODE`).
4.  **Run with Docker Compose**:
    ```bash
    docker-compose up -d --build
    ```
    This will start the application and a PostgreSQL database container.
